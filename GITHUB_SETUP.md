# GitHub Setup

## 1. Open the project folder

```powershell
cd c:\Users\GAGAN\Desktop\Krauss_simulation\quantum_simulation_portfolio
```

## 2. Initialize git

```powershell
git init -b main
git add .
git commit -m "Initial commit: quantum simulation portfolio"
```

## 3. Create a GitHub repository

Recommended repository name:

`quantum-simulation-portfolio`

## 4. Connect the local folder to GitHub

After creating an empty GitHub repo in the browser, run:

```powershell
git remote add origin https://github.com/YOUR_USERNAME/quantum-simulation-portfolio.git
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.

## Final repository link

```text
https://github.com/YOUR_USERNAME/quantum-simulation-portfolio
```
